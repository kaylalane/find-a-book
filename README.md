# Find-a-book

A React Goodreads clone

## Technologies

- Vite
- React
- React Router
- Express
- Node

## Project Structure

### Client

- /dist: build files
- /public: assets
- /src: where the application lives

### Server

- /db: mongodb connection
- /routes: API routes w/ Express
- index.js: server file
